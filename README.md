# :computer: Sistema de gestão dos horários dos Laboratórios e Salas de Aula 

## :wrench: Funcionalidades do Sistema:
- Cadastro dos professores utilizando:
  - Nome, Sobrenome, e-mail e senha.
- Redefinição de senha;
- Login a partir deste cadastro;
- Dashboard com as salas reservadas pelo professor;
- Dashboard com as salas disponíveis para reserva de acordo com a data e horário;
- Professor pode realizar o cancelamento da reserva;
- Apresentação de relatório:
  - Semanal, mensal, trimestral e semestral.
